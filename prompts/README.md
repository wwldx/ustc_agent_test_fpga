# Prompts

Store safe-to-publish prompt drafts and role definitions here.

## Rules

- Do not paste private system prompts from proprietary products.
- Do not include API keys, organization IDs, local secrets, or private paths.
- Every prompt should state its allowed tools, forbidden actions, verification loop, and output evidence.
- For hardware-adjacent prompts, default to dry-run and require human confirmation.
